Class Geojoin::Relation
In: geojoin.rb
Parent: Object

The Relation class computes the geometric relationship between two features, and provides the full set of DE-9IM predicates:

  • equals?
  • disjoint?
  • touches?
  • overlaps?
  • crosses?
  • within?
  • contains?
  • intersects?
